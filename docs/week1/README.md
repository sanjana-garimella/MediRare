# Week 1 (Undergrad-Friendly) - Start Here

This project is split into role-based Week 1 tasks. Pick the guide that matches your role and follow it step-by-step.

## Repo Link

`https://github.com/sanjana-garimella/MediRare`

## One Rule That Matters

Do **not** commit to `main`.

Create and work on **only one** of these branches:

- `week1/nlp`
- `week1/biomedical`
- `week1/cv`
- `week1/backend`
- `week1/integration`

## Quick Git Commands (Copy/Paste)

```bash
git clone https://github.com/sanjana-garimella/MediRare
cd MediRare

# Pick ONE branch name from the list above
git checkout -b week1/<your-role>

git branch --show-current
```
